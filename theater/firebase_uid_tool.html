Firebase UID 取得ツール

For H/MIX Theater · Admin Setup
Googleアカウントでログインして、自分のFirebase UID (28文字の英数字)を取得します。
取得したUIDは、FirestoreのadminsコレクションにドキュメントIDとして登録します。
確認: Firebase Console の Authentication で、Googleプロバイダが有効になっていること。
ローカルで開いている場合、URLが http://localhost:... であること (file:// では動きません)。

次のステップ:
1. 上の UID をコピー
2. Firebase Console → Firestore Database → データ タブ
3. 「+ コレクションを開始」 → ID: admins
4. 新しいドキュメントのID欄にUIDを貼り付け
5. フィールド追加: since (timestamp, 現在時刻) と name (string, 秋山裕和)
6. 「保存」 → 完了
