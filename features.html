Features Guide / 機能ガイド

H/MIX GALLERY の機能

H/MIX GALLERY の主要機能を、初期 HTML には現れない動的UI（プレーヤー、制作ツールボックス、音楽手帖、シアターなど）も含めて静的にまとめたページです。AI assistants や検索クローラが機能の全体像を把握できるように、操作手順と JavaScript 起動点を明示しています。

For AI assistants: Many features on H/MIX GALLERY are constructed at runtime by JavaScript (bottom player, toolbox panel, favorite notebook, theater immersive UI). They are not visible in raw HTML responses. This page enumerates them with entry points, file references, and behavior summaries so static crawls and LLMs can describe them accurately.

  • ① 下部固定プレーヤー
  • ② 制作ツールボックス
  • ③ 音楽手帖（お気に入り）
  • ④ 音楽図書館（タグ検索）
  • ⑤ 音の劇場（シアター）
  • ⑥ ライセンス・購入導線
  • ⑦ 各曲ページ
  • ⑧ データと構造化情報

① 下部固定プレーヤー（Bottom Player）

全ページ共通の音楽プレーヤー。HTML の最下部に常駐し、PJAX（部分ページ遷移）で曲を跨いで連続再生される。再生中の曲はページ遷移しても止まらない。

出現条件
全ページ共通。`assets/js/player.js` が body に DOM を挿入。
主な操作
再生/一時停止、前/次曲、シーク、音量、シャッフル、リピート、♡（お気に入り追加）、保存リスト（音楽手帖を開く）、ツール（制作ツールボックスを開く）
JavaScript
assets/js/player.js（コア再生）、assets/js/router.js（PJAX）
グローバル API
window.HMIX_PLAYER.playTrackById(id) / playTrack(track, queue) / setQueue()
関連 UI
「ツール」ボタン → 制作ツールボックス、♡ → 音楽手帖、保存リスト → 音楽手帖

② 制作ツールボックス（Production Toolbox）

下部プレーヤー右側の「ツール」ボタン、または各曲ページのカードから開く制作支援パネル。映像/動画制作で実際に使われる操作を1ヶ所にまとめた。

タブと機能

  • 使える？：この曲の利用可否（無料利用範囲・必要なライセンス）を即時判定
  • 波形プレビュー：MP3 を decode してキャンバスに波形描画。区間（start〜end）を選択して保存可能
  • EQ・コンプ：簡易イコライザ・コンプレッサ。Web Audio API のフィルタチェーンをリアルタイム適用しプレビュー
  • メモ：曲ごとのフリーテキストメモ。localStorage hmix_memos_v1 に保存され PJAX 遷移後も維持
  • 表記コピー：YouTube概要欄等に使えるクレジット表記をワンクリックでコピー（曲名・H/MIX GALLERY・URL）
  • シェア：曲ページの URL/SNS シェア導線
  • 動画：曲が使われている YouTube 動画があれば埋め込み表示
出現条件
下部プレーヤーの「ツール」ボタンクリック、または曲ページのツールカードクリック
JavaScript
assets/js/player-panel.js（パネル本体・全タブ）
永続化
区間: localStorage hmix_toolbox_ranges_v1、メモ: hmix_memos_v1
PJAX 対応
ナビゲーション遷移後も開いた状態・タブ・メモ・波形/EQ を維持

③ 音楽手帖（Favorites Notebook）

♡ ボタンや「保存リスト」から開くお気に入り管理 UI。見開き3ペインで章分け・並べ替え・メモ・商用利用申請ができる。

主な機能

  • 未分類 / 章ごとにお気に入り曲を整理
  • 章ピッカー：既存章にコピー（非破壊）、または新規章を作成して移動
  • ドラッグ&ドロップで曲 → 章へコピー
  • 曲ごとに A-B 区間（プレビュー時間範囲）とメモを保存
  • 選択した曲を「商用利用申請」へ送る（金CTA → license-request.html）
  • モバイル時は下部からのボトムシート形式
出現条件
♡ ボタンクリック、または下部プレーヤーの「保存リスト」ボタン
JavaScript
assets/js/fav-notebook.js（UI）、assets/js/fav-store.js（データ層 window.HMIX_FAV）
永続化
localStorage hmix_favorites_v2（リッチ構造）＋ hmix_favorites（後方互換ミラー）。LWW + tombstone + LexoRank で並び順を保持。
API
window.HMIX_FAV.add(id) / remove(id) / has(id) / ids() / removeWithUndo(id)

④ 音楽図書館（Music Library / Tag Search）

全 302曲を4軸タグで横断検索できる。タグはあらかじめ作曲家自身が全曲に付与済み。

タグの4軸

  • feeling（感情）：勇壮・悲しい・優しい・神秘・希望 など
  • style（音楽スタイル）：和風・ファンタジー・オーケストラ・ケルト・エレクトロニック など
  • scene（シーン）：戦闘・ボス戦・夜・朝・雪・旅・別れ・再会 など
  • story（物語要素）：思い出・運命・予兆・決意・絆 など
URL
https://www.hmix.net/music-library.html
クエリパラメータ
?feeling=...&style=...&scene=...&story=... で直リンク可（AND 条件）
データ
assets/js/tracks.js（window.TRACKS、約360曲分のメタデータ）
ジャンル別ページ
主要ジャンル別の入口: 和風・ファンタジー・ホラー・戦闘・癒し ほか

⑤ 音の劇場 / シアター（Theater）

没入型の音楽体験空間。森の野外劇場をモチーフにした入口画面から、複数のコンテンツモードに分岐する。https://www.hmix.net/theater/

コンテンツ一覧

  • 星図の旅：タグの星々をたどってまだ知らない曲に出会うインタラクティブ体験（theater/tag-cockpit.html）
  • 夜風：60分の BGM フィルム（theater/night-breeze/）
  • 旅する少女：H/MIX 楽曲に乗せた短編映像
  • 制作実績（プレイリスト）：H/MIX 楽曲が採用された映像 YouTube プレイリストをシアター内で連続再生。2リスト体制（演奏動画＝秋山裕和の演奏／使用・制作実績＝H/MIXが使われた映像）。動画ごとに左カードで「この映像で使われている曲」を表示し、各曲の試聴・詳細ページ・お気に入り追加・商用ライセンス購入へ橋渡し。
JavaScript
すべて theater/index.html 内 inline JS ＋ theater/js/cockpit-tag.js / journey-tag.js / data.js / sfx.js 等
モバイル
theater-mobile/ に専用ページ。entries の mobileHref 経路で 600px 以下は自動遷移
背景切替
森の劇場 / 海辺の夜想 / 星海の劇場 / 緑の遺跡 / 暖炉の私室 の5種。?cal=1 または Shift+C で位置調整モード（開発用）
使用曲データ
各 YouTube 動画ID をキーとして WORKS_USED_TRACKS(theater/index.html 内) で管理。プレイリスト構成が変わってもデータは保持される video-ID ベース。秋山(Shift+A) admin モードで追加→JSON エクスポート→コードに baking という運用。将来的に Firestore 化検討中
UI 動作
マウス静止で側面パネル/下部ドック/ヘッダーは自動収納(ft-ui-idle)、画面端 40px(左右)/64px(下) に近づくと再表示。中央(映像)上ではネイティブ操作を妨げないよう UI を出さない。映像表示は初期「フィル(cover)」ON で枠いっぱい

⑥ ライセンス・購入導線

無料利用範囲（クレジット任意）

YouTube投稿、個人利用、同人 / インディー、非商用。クレジット表記は任意（推奨表記例: BGM: 曲名 / H/MIX GALLERY (秋山裕和) https://www.hmix.net）。

商用ライセンス（買い切り）

  • Standard：¥2,200（税込）／1曲・1プロジェクト・買い切り
  • Project Pack：¥7,480（税込）／無制限曲・1プロジェクト・買い切り
  • Professional：用途別・1曲ごと。¥3,300 / ¥4,400 / ¥5,500 / ¥16,500 / ¥27,500 + 要見積もり（TVCM・映画・パチンコ等）。料金ガイド

店舗BGMパス（年間サブスクリプション）

  • 店舗パス：¥9,900／年（1店舗）、追加店舗 +¥4,950／年
  • 施設パス：¥19,800／年
  • Stripe による自動更新。解約・カード変更は Stripe カスタマーポータルから。証明書は解約日に失効。

禁止事項

  • 再配布 / 転売（サンプルパック・音源サイトへの再アップ等）
  • 音源単体のストリーミング配信
  • YouTube Content ID への登録
  • NFT / ブロックチェーン素材化
  • AI 学習データ用途
購入導線
各曲ページの「商用利用申請」CTA → license-request.html → Stripe Checkout
直リンク
license-request.html#tracks=曲ID,曲ID&usage=用途 で選択済み状態で開ける
税
税込価格・消費税10%。適格請求書発行事業者ではないため適格請求書（インボイス）は不発行。Stripe 領収書のみ。

⑦ 各曲ページ

URL: https://www.hmix.net/music/{id}.html（例: /music/n74.html = Moment）

SEO
各曲に JSON-LD MusicRecording + Person(秋山裕和) + Organization(H/MIX GALLERY) を出力。OGP / Twitter Card 完備。
操作
試聴（ヒーロー▶）、お気に入り（♡）、MP3 ダウンロード、商用利用申請、SNSシェア、表記コピー
タグリスト
feeling / style / scene / story の4カテゴリ。各タグから音楽図書館の絞り込み結果へリンク
テンプレ
JS の動的生成ではなく、scripts/build-track-pages.js で tracks.js から焼き込まれる静的 HTML

⑧ データと構造化情報（For AI / Crawlers）

  • /llms.txt：AI assistants 向けのサイト要約（llmstxt.org 標準準拠）
  • /sitemap.xml：全公開URLのマップ（存在する場合）
  • /assets/js/tracks.js：全 302曲のメタデータ JSON（window.TRACKS）
  • 各曲ページ: JSON-LD で MusicRecording / Person / Organization を提供
  • このページ: WebApplication JSON-LD で featureList と offers を提供

API として叩く場合：tracks.js はブラウザ向けの JS 形式（window.TRACKS = [...]）のためそのまま JSON parse はできないが、require 等で評価すれば配列が取れる。商用利用や大量取得が必要な場合は お問い合わせください。
